# Python DSA Practice

A collection of Data Structures & Algorithms (DSA) solutions implemented in Python, sourced from [LeetCode](https://leetcode.com/).

I solve **at least Two DSA Problem daily** and update this repository every day to track my progress and build consistent problem-solving habits.

## Folder Structure

Each LeetCode problem is organized into its own directory named after the problem number and slug (e.g., `0001-two-sum/`). Inside each directory:

- **`<problem-slug>.py`** — Python solution file
- **`README.md`** — Problem description (title, difficulty, examples, constraints)

## Usage

To run any solution, navigate to its directory and execute the Python file:

```bash
python <problem-folder>/<problem-file>.py
```

For example:

```bash
python "0001-two-sum/0001-two-sum.py"
```

Alternatively, you can run from the root:

```bash
python -m "0001-two-sum.0001-two-sum"
```

> **Note:** Some solutions may require additional imports (e.g., `List` from `typing`). Those are included in the solution files.

## Requirements

- Python 3.x

No external libraries are needed.
